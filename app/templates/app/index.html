How-to tail?

Install App

In ubuntu:

In ubuntu you need to add ppa repository and install package

sudo add-apt-repository nvbn-rm/ppa
sudo apt-get update
sudo apt-get install tailme

In other Linux

In other linux distros you can manualy build package or install via pip

pip install -e git+https://github.com/nvbn/cantailme-client.git#egg=tailme

Use It

You only need run app with tailme pipe, and session page open in your browser!

do-long-app | tailme

Or you can use silent mode and copy link from terminal

do-long-app | tailme --silent

About CanTail.Me

This is simple servise for providing tail -f over network.

Or you can use it like paste service alternative with terminal integration.

It's free software and you can get client and server side source from github.
